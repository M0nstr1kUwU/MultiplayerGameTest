# Multiplayer Soul Knight 3D

Мультиплеерный 3D-прототип на:

- Three.js
- Socket.IO
- Fastify
- PostgreSQL

## Что есть в v7

- регистрация и вход через серверные `HttpOnly`-сессии;
- PostgreSQL для пользователей, сессий и глобального PvE-лидерборда;
- меню игрока со стилем персонажа, списком лобби, входом по коду и глобальным PvE топ-10;
- красивые короткие коды лобби формата `ABCD-1234`;
- лобби с лидером, а не глобальным админом;
- фиксированный режим лобби после создания;
- PvE-сетка комнат в стиле Soul Knight;
- закрытая босс-комната;
- комната снабжения и оружейные комнаты;
- портал с голосованием всех живых игроков;
- выбор прокачек после уровня;
- воскрешение друзей через удержание `F`;
- PvP до заданного количества побед;
- 3 слота оружия;
- патроны, перезарядка без выбрасывания обоймы, выбрасывание оружия на `G` и передача патронов через `Q + 1..4`;
- уровни оружия отображаются прямо в названии;
- временная стилизация игроков/врагов процедурными формами;
- цвет персонажа в меню;
- Tab-список игроков и кик от лидера;
- мини-карта с мостиками между комнатами;
- телеграфы атак боссов исчезают сразу после удара;
- двери сделаны как двойные замковые врата, раскрывающиеся из центра наружу;
- документация по ассетам, 64x64 иконкам, текстурам, Blender-анимациям и Render.com.

## Запуск локально

Нужен PostgreSQL. Самый простой вариант — через Docker:

```bash
cp .env.example .env
npm install
npm run dev
```

Docker для локального запуска больше не обязателен: по умолчанию используется локальная JSON-база `server/.data/local-db.json`. Для PostgreSQL поставь `DB_DRIVER=postgres` и `DATABASE_URL` в `.env`.

Клиент:

```text
http://localhost:5173
```

Сервер:

```text
http://localhost:3000
```

## Управление по умолчанию

| Действие | Кнопка |
|---|---|
| Движение | WASD |
| Стрельба / атака | ЛКМ или Space |
| Подобрать / открыть / голосовать у портала | E |
| Воскрешение | F удерживать 5 секунд |
| Перезарядка | R |
| Выкинуть оружие | G |
| Выкинуть пачку патронов light/shell/energy/heavy | Q + 1 / 2 / 3 / 4 |
| Способность | Left Shift |
| Слоты оружия | 1 / 2 / 3 |
| Список игроков | Tab |
| Настройки | Esc |
