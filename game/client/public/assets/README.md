# Assets folder

Все файлы из `client/public` доступны в браузере от корня сайта.

Пример:

```text
client/public/assets/items/weapons/rusty_pistol.png
```

В коде это будет:

```text
/assets/items/weapons/rusty_pistol.png
```

См. подробности в `docs/ASSETS_AND_MODELS.md` и `docs/ANIMATION_GUIDE.md`.


## Размеры иконок

Иконки оружия, предметов, обойм и расходников: **64x64 PNG** с прозрачным фоном.

- `assets/items/weapons/*.png` — иконка в инвентаре.
- `assets/items/ground/*.png` — вид предмета на земле.
- `assets/items/consumables/*.png` — обоймы, ресалки, лечение и другие расходники.
